Send Metadata

Send Event Metadata()

topic_str                 : Enter the topic of event metadata.
source_key                : source key of event metadata.
source_value              : source value of event metadata.
data_key                  : data key of event metadata.
data_value                : data value of event metadata.
element_flag              : (Option) element data usage flag of event metadata.
element_name              : (Option) name of element.
element_descriptor_name   : (Option) descriptor name of element.
element_type_name         : (Option) type name of element.
element_likelihood        : (Option) likelihood of element.(float 0 ~ 1)

Send Frame Metadata()

object_id    : Enter the object id of frame metadata.
parent_id    : parent id of frame metadata.
likelihood   : likelihood of frame metadata.(float 0 ~ 1)
category     : category of frame metadata.(Reference: ObjectCategory,0 ~ 14)
rect_sx      : starting point of x.
rect_ex      : ending point of x.
rect_sy      : starting point of y.
rect_ey      : ending point of y.
