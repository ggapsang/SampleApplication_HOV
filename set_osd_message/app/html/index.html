Set OSD Message

SetOSDMessage()

Please go to the link below to check the parameters defined in the protocol attribute file. 
http:///stw-cgi/attributes.cgi
Enable             : Enter the Enable OSD message.
OSD Color          : color of the OSD message.(White/Black/Gray)
Transparency       : transparency of the OSD message.(Off/Low/Medium/High)
OSD                : OSD message.
OSD Position Settings
Set OSDPosition:
OSD Position       : Position of the OSD message.(TopLeft/TopRight/BottomLeft/BottomRight)
Set PositionX and PositionY:
Position X         : Position X of the OSD message.(min ~ max of PositionX)
Position Y         : Position Y of the OSD message..(min ~ max of PositionY)
OSD Font Size Settings
Set CustomFontSize:
Custom Font Size   : size of the OSD message.(min ~ max of CustomFontSize)
Set FontSize:
Font Size          : size of the OSD message.(Small/Medium/Large)
