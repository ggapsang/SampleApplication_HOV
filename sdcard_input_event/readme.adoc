== sdcard_input_event

This sample application is designed to test the SD-card-input events.
For more details, refer to *1.3 SD Card Events* in
*Hanwha_Vision_Open_Platform_SDK_API*.

Related events are as follows:

*Receivable Events*

* 1.3 SD Card Events +
You can receive SD card-related events. This event alerts you when the
SD card is inserted or when no recording is taking place. To use this
event, allow the application to access the SD card.
** SD Card Inserted
** SD Card Removed
* Event
+
[cols=",,",options="header",]
|===
|Event type |Header file |Event type(enum)
|SD Card Inserted |i_device_io_manager.h
|IDeviceIoManager::EEventType::eSdcardInserted
|SD Card Removed |i_device_io_manager.h
|IDeviceIoManager::EEventType::eSdcardInserted
|===

* Parameters
* SerializableString
+
[cols=",,",options="header",]
|===
|Variable |Type |Description
|response |string(Json) |SD card slot number and block number
|===
+
[source,json]
----
{"sd_index":1,"block_index":0}
----

=== Prerequisites

* SD card compatible device
* Hanwha Vision Open Platform 23.00.00 or higher

=== Scenario

* Preconditions
[arabic]
. Open the camera web viewer, go to [Setup]>[Open Platform].
. Click [Install] to install the application.
. Click [Start] to start the application.
* Test for SD card inserted
[arabic]
. Insert the SD card into the SD card slot.
. SampleComponent receives [SD Card Inserted] event from the camera.
. Check console.log.
* Test for SD card removed
[arabic]
. Remove the inserted SD card from the slot.
. SampleComponent receives [SD Card Removed] event from the camera.
. Check console.log.

=== Result

Test for SD card inserted

....
Index   Status
0   inserted
....

Test for SD card removed

....
Index   Status
0   removed
....

=== Building Application

[arabic]
. Build application.
+
....
$ APP_NAME=sdcard_input_event SDK_VER=24.06.14(your SDK version) SOC=[cv5, orinnx8g_jp512] docker compose up
$ docker compose down --remove-orphans
....
. Check the build results in current directory. If successful, you will be able to find the cap file.
